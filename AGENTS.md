
## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues (gh CLI). External PRs are not a triage surface. See docs/agents/issue-tracker.md.

### Triage labels

Labels use Chinese names: 待评估 / 待确认 / AI处理 / 人工处理 / 不处理. See docs/agents/triage-labels.md.

### Domain docs

Single-context layout: CONTEXT.md at root + docs/adr/ for architecture decisions. See docs/agents/domain.md.
